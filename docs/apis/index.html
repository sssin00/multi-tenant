API 문서 허브

서비스별 API 문서는 docs/apis/{service-name} 아래에 HTML로 관리합니다. 각 서비스의 index.html은 API 목록을 보여주고, 개별 endpoint는 별도 HTML 문서로 분리합니다.

작성 규칙

서비스별 목록docs/apis/{service-name}/index.html에 해당 서비스의 API 목록을 둡니다.
Endpoint 상세docs/apis/{service-name}/{method}-{resource}.html 형식으로 상세 문서를 둡니다.
예시docs/apis/gateway-service/get-health.html
연결새 API 문서를 만들면 이 허브와 서비스 개발과정 문서에서 함께 연결합니다.
PostmanAPI를 만들거나 추가/수정/삭제하면 postman/multi-tenant-local.postman_collection.json도 같은 변경으로 갱신합니다.
