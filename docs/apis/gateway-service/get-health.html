gateway-service API

GET /health

gateway-service 프로세스가 살아 있고 HTTP 요청에 응답할 수 있는지 확인합니다.

GET /health PUBLIC

기본 정보

목적ALB와 운영 모니터링에서 gateway-service 생존 상태를 확인합니다.
인증없음. Health check 대상이므로 public endpoint로 둡니다.
외부 의존성확인하지 않습니다. gateway 설정, JWT, proxy route URL, rate limit, security/CORS 준비 상태는 /ready에서 다룹니다.
공통 HeaderX-Request-Id와 보안 header가 응답에 포함됩니다.
성공 상태200 OK
