gateway-service API 문서

gateway-service에 추가되는 API를 누적 기록합니다. 목록에는 기본 정보만 두고, 상세는 각 API 문서에서 펼쳐 확인합니다.

API 목록

GET

/health

gateway-service 프로세스 생존 확인용 API입니다.

ACTIVE GET

/ready

설정, JWT, proxy route, rate limit, security/CORS 기준을 확인하는 readiness API입니다.

READY ANY

/api/auth/** · /api/admin/** · /api/app/**

auth, admin, app route group을 각각 Auth/IAM, Admin BFF, User BFF로 전달합니다. tenant-service, wms-service, audit-log-service는 BFF 또는 내부 worker 경로에서 사용합니다.

ACTIVE

API 문서 허브로 돌아가기 · gateway-service 개발과정으로 돌아가기 · 문서 대시보드로 돌아가기
