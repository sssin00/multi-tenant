Development Process

wms-service 개발과정

WMS 기준정보, 재고, 입고, 출고, 작업 지시를 담당하는 첫 메인 도메인 서비스입니다.

서비스/앱 역할

위치apps/wms-service
카테고리Backend Service · Domain Service
초기 구조하나의 서비스 내부를 모듈형 모놀리스로 구성
도메인 순서warehouse → location → item → inventory → inbound → outbound → task → shipment
이벤트wms.inventory.adjusted, wms.inbound.confirmed, wms.outbound.allocated, wms.outbound.shipped

기능 정의

기준정보warehouse, location, item, partner, unit, barcode 등 WMS 운영 기준정보를 tenant 범위로 관리합니다.
재고 관리현재고/가용재고 조회, 재고 이동, 재고 조정, lot/serial 정책과 관련 audit/outbox 기록을 담당합니다.
입고 관리입고 예정, 검수, 적치 완료 흐름과 입고 확정 이벤트 저장을 담당합니다.
출고 관리출고 요청, 재고 할당, 피킹, 출하 완료 흐름과 출고 이벤트 저장을 담당합니다.
권한/테넌트중요 업무 action 실행 전 tenantId 조건과 필요한 permission/scope를 검증합니다.
제외 범위화면 데이터 조합, 사용자 인증, tenant lifecycle 관리, 외부 ERP 재시도 orchestration은 직접 소유하지 않습니다.

구현 상태

상태착수 전
선행 조건tenant context, Auth/IAM permission check, WMS 1차 domain scope가 확정되어야 합니다.
API 문서WMS API가 생기면 docs/apis/wms-service/index.html에 계약을 기록합니다.

남은 작업

기능별 작업 내역

기준정보warehouse, location, item, partner, unit, barcode를 tenantId 조건으로 관리합니다.
재고현재고, 가용재고, 이동, 조정, lot/serial 정책과 audit 대상 변경을 구현합니다.
입고입고 예정, 검수, 적치 완료 흐름과 wms.inbound.confirmed event를 저장합니다.
출고출고 요청, 할당, 피킹, 출하 완료 흐름과 wms.outbound.allocated, wms.outbound.shipped event를 저장합니다.

테스트 시나리오

WMS-TENANT-001tenantId가 다른 데이터는 목록/상세/변경 API에서 조회되거나 수정되지 않는다.
WMS-INV-001재고 조정은 재고 수량 변경, audit log, outbox event를 같은 흐름에서 확인할 수 있다.
WMS-OUT-001출고 할당은 가용 재고 부족 시 표준 error code로 실패한다.

후속 작업

현재 WMS 1차 구현 이후 이어서 진행할 작업입니다. 이 표의 작업이 진행되면 관련 코드와 문서를 함께 갱신합니다.

우선순위 작업 범위 완료 기준 함께 갱신할 문서
P0 ERP 연동 기준 입출고 확정 이후 ERP 전송, 실패 retry, 수동 재전송 기준을 별도 문서로 확장합니다. ERP 전송 성공/실패와 재시도 상태가 requestId, tenantId, audit/outbox evidence로 추적됩니다. wms-service 개발 문서, events/outbox 규칙, ERP 연동 문서
P1 고급 재고 정책 lot/serial, FEFO/FIFO, 유통기한, 보류 재고, 가용 재고 계산 정책을 단계적으로 추가합니다. 재고 할당과 조정이 tenant별 정책을 따르고 교차 tenant 접근 없이 검증됩니다. wms-service API 문서, multi-tenant 규칙, testing 규칙
P1 출고 작업 고도화 wave picking, task assignment, 작업자별 picking/packing 흐름은 1차 API 안정화 후 추가합니다. 출고 할당부터 작업 완료까지 상태 전이가 audit log와 event로 남습니다. wms-service 개발 문서, web-wms 개발 문서, events/outbox 규칙
