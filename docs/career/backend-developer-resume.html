Multi-Tenant Smart Factory Platform

Backend / Platform Developer · 개인 프로젝트

2026.06 - 진행 중
  • AWS ECS Fargate 기반 멀티테넌트 스마트 팩토리 플랫폼을 WMS first 전략으로 설계하고 구현했습니다.
  • gateway-service를 외부 API 단일 진입점으로 두고, Admin/User BFF가 화면 조합과 tenant readiness 확인을 담당하도록 아키텍처를 분리했습니다.
  • auth-iam-service에서 JWT access token, opaque refresh token rotation, RBAC, permission summary/check API, system_admin tenantless login을 구현했습니다.
  • tenant-service에서 tenant/module/domain/status/readiness API를 구현하고, business API가 tenant 상태를 기준으로 동작하도록 연계했습니다.
  • wms-service에서 창고/로케이션/품목, 재고 원장/잔고, 재고 조정, 일 단위 스냅샷, 입고 확정, 출고 할당/포장/출하 API를 구현했습니다.
  • outbox-relay-service와 audit-log-service를 통해 업무 이벤트 발행, SQS/DLQ 기반 전달, append-only 감사 로그 저장 구조를 구성했습니다.
  • CDK TypeScript로 ECR, ECS service/task, ALB route, Cloud Map, Redis, SQS/DLQ, CloudWatch log group, Security Group을 관리하도록 구성했습니다.

현재 저장소 기준 CDK backend service 8개, 서비스별 API 문서 12개, 시나리오 기반 검증 로그 32개를 유지하고 있습니다.
